## 需求背景

前端开发者在日常工作中频繁需要处理图片资源：压缩体积以优化加载性能、转换格式以适配不同平台、批量管理 SVG 图标等。目前这些操作分散在多个在线工具（TinyPNG、Squoosh、SVGOMG 等）中，存在以下痛点：

1. **在线工具有使用次数限制**，部分需要付费
2. **隐私安全顾虑**，图片需上传到第三方服务器
3. **无法批量处理**，大量文件时效率极低
4. **功能分散**，需要在多个网站间切换

本项目旨在构建一个 **Electron 桌面端一站式图片工具库**，让前端开发者在离线环境下完成所有图片处理工作。

## 功能描述

### 核心功能（MVP 优先级排序）

#### P1：SVG 批量查看与管理

整个 SVG 图标库的可视化浏览与批量操作中心。

| 功能点       | 描述                                                     |
| ------------ | -------------------------------------------------------- |
| 双视图浏览   | 支持宫格缩略图与列表视图两种布局，可自由切换             |
| 文件名搜索   | 支持按文件名实时搜索筛选 SVG                             |
| 批量颜色修改 | 统一修改选中 SVG 的 fill / stroke 颜色，适配不同 UI 主题 |
| 多倍率导出   | 支持将 SVG 导出为 PNG @1x / @2x / @3x                    |
| 文件夹加载   | 支持拖入或选择文件夹批量加载 SVG 文件                    |

#### P2：图片压缩

支持主流图片格式的高质量压缩，提供多种压缩策略。

| 功能点   | 描述                                                                             |
| -------- | -------------------------------------------------------------------------------- |
| 支持格式 | JPEG、PNG、WebP、GIF、AVIF、TIFF                                                 |
| 压缩模式 | 有损压缩（可调质量百分比）、无损压缩、智能推荐（自动选最优参数），用户可自由选择 |
| 批量压缩 | 支持一次拖入多张图片或选择整个文件夹                                             |
| 预览对比 | 压缩后提供原图 vs 压缩图的并排画质对比 + 文件大小对比                            |
| 输出策略 | 用户可选择：覆盖原文件 / 导出到指定目录 / 原文件名追加后缀                       |

#### P3：格式转换

支持所有主流图片格式的全面互转。

| 功能点     | 描述                                                                                           |
| ---------- | ---------------------------------------------------------------------------------------------- |
| 支持格式   | JPEG ↔ PNG ↔ WebP ↔ AVIF ↔ TIFF ↔ BMP ↔ ICO，SVG → PNG/JPEG 栅格化，GIF ↔ APNG ↔ WebP 动图互转 |
| 尺寸预设   | 提供常用预设尺寸（favicon 16×16、App Icon 1024×1024 等），支持自定义                           |
| 批量转换   | 一键将多张图片统一转为目标格式，支持统一参数配置                                               |
| ICO 多尺寸 | ICO 格式支持嵌入多种尺寸（16×16、32×32、48×48、256×256）                                       |

### 功能边界

- ✅ 包含：图片处理、格式转换、SVG 管理、辅助工具
- ✅ 包含：批量操作、拖拽交互、快捷键
- ❌ 不包含：在线上传/云存储
- ❌ 不包含：图片 AI 处理（如 AI 修图、超分辨率）
- ❌ 不包含：视频处理

## 用户故事

- 作为**前端开发者**，我希望**批量查看项目中的所有 SVG 图标并按文件名搜索**，以便**快速找到需要的图标资源**
- 作为**前端开发者**，我希望**一键把选中的 SVG 颜色统一修改为当前主题色**，以便**快速适配深色/浅色主题**
- 作为**前端开发者**，我希望**将 SVG 导出为 @1x @2x @3x 多倍率 PNG**，以便**在移动端项目中使用位图资源**
- 作为**前端开发者**，我希望**拖入一个文件夹批量压缩所有图片，并实时看到压缩前后对比**，以便**在上线前优化网站加载性能**
- 作为**前端开发者**，我希望**一键将 100 张 PNG 批量转为 WebP 格式并使用预设参数**，以便**减少构建产物体积**
- 作为**前端开发者**，我希望**用 Ctrl+V 直接粘贴剪贴板中的截图进行压缩或格式转换**，以便**简化日常工作流**

## 业务规则

| 编号 | 规则             | 说明                                                                           |
| ---- | ---------------- | ------------------------------------------------------------------------------ |
| BR01 | 压缩质量范围     | 有损压缩质量范围 1%~100%，默认 80%                                             |
| BR02 | 智能推荐算法     | 根据图片类型和内容自动选择最优压缩参数，目标在画质损失最小化前提下最大化压缩率 |
| BR03 | 批量操作文件数量 | 单次批量操作无硬性文件数量限制，但需展示处理进度                               |
| BR04 | SVG 颜色修改     | 仅修改 SVG 中的 fill/stroke 属性，不改变 SVG 结构                              |
| BR05 | 多倍率导出       | @1x 为原始尺寸，@2x 为 2 倍，@3x 为 3 倍                                       |
| BR06 | ICO 多尺寸       | 默认嵌入 16×16、32×32、48×48、256×256 四种尺寸                                 |
| BR07 | 离线运行         | 所有功能必须在无网络环境下正常工作                                             |

## 验收标准

### SVG 批量查看（P1）

- [ ] AC01: 可通过拖拽或文件选择器加载 SVG 文件/文件夹
- [ ] AC02: 宫格视图与列表视图可自由切换，状态保持
- [ ] AC03: 输入关键词可实时过滤显示匹配的 SVG
- [ ] AC04: 选中多个 SVG 后可一键修改 fill/stroke 颜色
- [ ] AC05: 可将 SVG 导出为 @1x @2x @3x PNG，文件命名正确

### 图片压缩（P2）

- [ ] AC06: 支持 JPEG/PNG/WebP/GIF/AVIF/TIFF 六种格式的压缩
- [ ] AC07: 可在有损、无损、智能推荐三种模式间切换
- [ ] AC08: 压缩后可并排对比原图与压缩图的画质和文件大小
- [ ] AC09: 批量压缩时显示每个文件的处理进度
- [ ] AC10: 输出位置支持覆盖原文件、指定目录、追加后缀三种策略

### 格式转换（P3）

- [ ] AC11: 支持所有规定格式间的互转
- [ ] AC12: 转换时可选择预设尺寸或自定义尺寸
- [ ] AC13: 批量转换支持统一参数配置
- [ ] AC14: ICO 格式输出包含多种尺寸

### 通用

- [ ] AC15: 支持文件/文件夹拖拽到窗口
- [ ] AC16: 支持 Ctrl+V 粘贴剪贴板图片
- [ ] AC17: 暗色模式可正常切换
- [ ] AC18: 所有功能离线可用

## 边界与异常

| 场景             | 触发条件           | 处理方式                                             |
| ---------------- | ------------------ | ---------------------------------------------------- |
| 不支持的文件格式 | 拖入非图片文件     | 提示"不支持的文件格式"，忽略该文件并继续处理其他文件 |
| 文件过大         | 单张图片超过 100MB | 提示警告，用户确认后继续处理                         |
| 压缩失败         | 图片损坏或编码异常 | 标记失败状态并展示错误原因，不阻断其他文件           |
| 磁盘空间不足     | 输出目录空间不足   | 中止操作并提示用户释放空间                           |
| SVG 解析失败     | SVG 文件格式不合规 | 在列表中标记为"无法解析"，显示文件名但不渲染预览     |
| 批量操作中断     | 用户手动取消       | 已完成的文件保留结果，未处理的文件保持原状           |
