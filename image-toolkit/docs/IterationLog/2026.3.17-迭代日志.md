## 2026-03-17

### 🆕 新增功能

- SVG 大图预览弹窗（缩放/平移/三种背景/源码查看/一键复制）
- PNG 导出对话框（倍率1x/2x/3x + 自定义像素宽高 + 锁定比例）
- SVG 批量 ZIP 打包下载
- 格式转换自定义宽高（锁定比例联动）
- 格式转换文件夹递归加载
- 全局快捷键 Ctrl+Z 撤销 / Ctrl+Shift+Z 重做 / Ctrl+A 全选 / Delete 删除
- Web Audio API 操作音效（成功/错误/删除/完成四种提示音，可关闭）

### 🔧 功能优化

- 文件列表追加模式 + 路径去重 + Toast 提示
- 输出目录选择 + electron-store 路径持久化
- 保留原文件开关（默认开启）
- 压缩/转换完成后"打开输出文件夹"按钮
- 格式兼容提示（源=目标时弹窗确认）
- 失败文件重试按钮 + 错误详情 Tooltip
- 清空按钮快捷键提示 Tooltip

### 🎨 UI/UX 改进

- CSS 变量全局主题适配（浅色/深色双主题）
- TransitionGroup 列表入退场动画
- FileListItem hover 高亮 + 阴影
- 压缩结果迷你柱状图对比
- 文件格式专属颜色标签（PNG蓝/JPG橙/WebP绿）
- SVG 选中 ✓ 角标 + 选中计数
- 骨架屏加载占位（脉冲动画）
- 空状态拖拽浮动动画指引
