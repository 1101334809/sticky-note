## 修复记录

在测试过程中发现并修复了以下问题：

| 问题                                    | 根因                                                                                 | 修复方案                                                                                 |
| --------------------------------------- | ------------------------------------------------------------------------------------ | ---------------------------------------------------------------------------------------- |
| `ERR_REQUIRE_ESM` 启动报错              | `createRequire(import.meta.url)` 在 Vite 打包后无法正确载入与 ESM/CJS 互操作冲突的包 | 改为标准 `import`（Vite 打包处理 interop），仅 pdf-parse 保留 `createRequire`+`external` |
| pdf-parse `\"default\" is not exported` | pdf-parse 是纯 CJS 包无 ESM default export                                           | 改为惰性 `createRequire` + 添加到 rollupOptions.external                                 |
