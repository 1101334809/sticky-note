## TC-001: 侧栏入口与页面导航

| 步骤 | 操作             | 预期结果                                   |
| ---- | ---------------- | ------------------------------------------ |
| 1    | 启动应用         | 侧栏显示「文档转换」菜单项                 |
| 2    | 点击「文档转换」 | 页面跳转到 /doc-convert                    |
| 3    | 确认页面标题     | 显示「文档转换」标题和描述文字             |
| 4    | 确认格式选择器   | 显示源格式和目标格式两个下拉框             |
| 5    | 确认默认格式     | 源格式默认 Word (.docx)，目标默认 Markdown |

---

## TC-002: 格式选择器联动

| 步骤 | 操作                        | 预期结果                                |
| ---- | --------------------------- | --------------------------------------- |
| 1    | 源格式选择 Word (.docx)     | 目标格式下拉显示: PDF / Markdown / HTML |
| 2    | 源格式切换为 PPT (.pptx)    | 目标格式下拉显示: Word / PDF / Image    |
| 3    | 源格式切换为 PDF (.pdf)     | 目标格式下拉显示: Word / Markdown       |
| 4    | 源格式切换为 Markdown (.md) | 目标格式下拉显示: HTML / Word           |
| 5    | 源格式切换为 HTML (.html)   | 目标格式下拉显示: Markdown              |
| 6    | 切换源格式后检查文件列表    | 文件列表自动清空                        |

---

## TC-003: 文件添加（拖拽 + 点击）

| 步骤 | 操作                                | 预期结果                             |
| ---- | ----------------------------------- | ------------------------------------ |
| 1    | 点击拖拽区域                        | 弹出文件选择对话框                   |
| 2    | 选择一个 .docx 文件                 | 文件列表显示该文件名，状态为「等待」 |
| 3    | 再次选择同一文件                    | 不重复添加（去重）                   |
| 4    | 拖拽 2 个 .docx 文件到页面          | 文件列表新增 2 项                    |
| 5    | 拖拽一个 .pdf 文件（源格式为 docx） | 提示「请拖入 Word 格式的文件」       |
| 6    | 点击文件条目的删除按钮              | 该文件从列表中移除                   |
| 7    | 点击「清空列表」                    | 所有文件清空                         |

## TC-004: Word → Markdown 转换

| 步骤 | 操作                                   | 预期结果                                            |
| ---- | -------------------------------------- | --------------------------------------------------- |
| 1    | 源 Word，目标 Markdown                 | 格式设置正确                                        |
| 2    | 添加含标题/列表/表格/图片的 .docx 文件 | 文件列表显示                                        |
| 3    | 点击「开始转换」                       | 状态变为「转换中」，进度条显示                      |
| 4    | 转换完成                               | 状态变为「完成」，显示绿色对勾                      |
| 5    | 检查输出 .md 文件                      | 标题层级保留、列表格式正确                          |
| 6    | 检查图片                               | 图片提取到 `文档名_images/` 目录，MD 中使用相对路径 |
